A program is to graphically spell out words, phrases or sentencess in semaphore.

The "Escape" key will hide and show the top bar.

The string field will let the user input a string to be spelled out. The "Change String" button will change the string that is spelled out.

There are two modes. "String Mode" will repeatively spell out the string that was inputted. "Random Mode" will randomly display letter in the alphabet.

The "Speed" is the duration between the letters changing. Speed is measured in millaseconds. The "Slower" button will increase the duration by 100 ms. The "Faster" will decrease the duration by 100 ms. The lowest duration is 500 ms.
